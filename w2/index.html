Introduction to HTML Elements

Meta Tags

Meta tags provide information about the HTML document. They are not displayed on the page but are used by browsers and search engines.

HTML Entities

HTML entities are used to display special characters. For example:

< represents the less-than sign

> represents the greater-than sign

© represents the copyright symbol
