HTML and CSS Training

1. Lists

Unordered Lists

  • Used for non-sequential items
  • Created with <ul> tag
  • List items use <li> tag

Ordered Lists

  1. Used for sequential items
  2. Created with <ol> tag
  3. Can use type attribute for different numbering styles

Description Lists

Term (dt)
Description (dd)

2. Tables

Tables are used to display data in rows and columns.

Tag Description
<table> Defines a table
<thead> Defines a table header section
<tbody> Defines a table body section
<tfoot> Defines a table footer section
<tr> Defines a table row
<th> Defines a table header
<td> Defines a table cell

3. Containers

Containers are used to group and structure content:

  • <div>: Generic container for flow content
  • <span>: Generic inline container
  • <figure>: Specialized container for grouping media and captions

4. Semantic Elements

Semantic elements provide meaning to the structure of web content:

  • <header>
  • <nav>
  • <main>
  • <article>
  • <section>
  • <aside>
  • <footer>

5. Structuring Web Pages

A basic structure for a web page:

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Page Title</title>
</head>
<body>
    <header>
        <nav>
            <!-- Navigation menu -->
        </nav>
    </header>
    <main>
        <article>
            <section>
                <!-- Content sections -->
            </section>
        </article>
        <aside>
            <!-- Sidebar content -->
        </aside>
    </main>
    <footer>
        <!-- Footer content -->
    </footer>
</body>
</html>

6. CSS (Cascading Style Sheets)

CSS is used to style HTML elements. There are three ways to include CSS:

  1. Inline CSS: Using the style attribute
  2. Internal CSS: Using the <style> tag in the <head> section
  3. External CSS: Linking an external .css file

Example of CSS properties:

.example-class {
    color: blue;
    font-size: 16px;
    margin: 10px;
    padding: 5px;
    background-color: #f0f0f0;
}
