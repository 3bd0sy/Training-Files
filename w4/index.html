Comprehensive CSS Guide with Examples

1. Basic CSS

CSS allows you to style HTML elements with various properties such as colors, fonts, and layouts.

Example:

p {
  color: blue;
  font-size: 16px;
  font-family: Arial, sans-serif;
}

This is a styled paragraph.

2. Borders

Borders can be added to elements with different styles such as solid, dotted, and double. You can also change the border color, width, and radius.

Example:

.box {
  border: 2px solid red;
  border-radius: 10px;
  padding: 10px;
}
This is a box with a border.

3. Normalize CSS

Normalize.css is a small CSS file that ensures cross-browser consistency by resetting browser styles.

Example:

To use Normalize.css, include this link in your HTML head:

<link rel="stylesheet" href="https://cdnjs.cloudflare.com/ajax/libs/normalize/8.0.1/normalize.min.css">

4. CSS Selectors

Class and ID Selectors

  • Class selector: .classname
  • ID selector: #idname

Attribute Selectors

  • Exact match: [attribute="value"]
  • Contains: [attribute~="value"]

Example:

.highlight { background-color: yellow; }
#unique { font-weight: bold; }

This is highlighted text.

This is unique text.

5. Relational Selectors

  • Descendant: parent child
  • Child: parent > child
  • Adjacent Sibling: element + sibling
  • General Sibling: element ~ siblings

Example:

div p { color: blue; }
ul > li { list-style-type: square; }
h1 + p { font-weight: bold; }
h1 ~ p { font-style: italic; }

This paragraph is blue (descendant selector).

6. Pseudo-classes

  • :hover - Triggered when hovering over an element.
  • :nth-child(n) - Selects the nth child element.

Example:

button:hover { background-color: lightblue; }
li:nth-child(odd) { background-color: #f2f2f2; }

7. Pseudo-elements

  • ::before - Adds content before an element.
  • ::after - Adds content after an element.
  • ::first-line - Styles the first line of an element.

Example:

p::before { content: "→ "; color: red; }
p::after { content: " ←"; color: blue; }
p::first-line { font-weight: bold; }

8. Selector Specificity

Specificity determines which style will be applied when multiple rules target the same element.

  • Inline styles have the highest specificity.
  • ID selectors are stronger than class selectors.

Example:

#myId { color: red; }
.myClass { color: blue; }
p { color: green; }

This text is red due to ID selector specificity.

9. Inheritance

Some CSS properties are inherited from parent elements, such as color and font-family.

Example:

div {
  color: blue;
  font-family: Arial, sans-serif;
}

10. Colors and Gradients

Color Formats

  • RGB: rgb(255, 0, 0)
  • Hexadecimal: #FF0000

Gradients

Gradients can create smooth transitions between colors.

Example:

.color-example { color: rgb(0, 128, 0); }
.gradient-example {
  background: linear-gradient(to right, red, yellow);
  height: 50px;
}

This text uses RGB color.
