Topic Description Example
Basic CSS CSS allows styling HTML elements. p { color: blue; font-size: 16px; }
Borders Borders can be added to the elements. .box { border: 2px solid red; }
CSS Selectors Include class, ID, attribute, and sibling selectors. #unique { font-weight: bold; }
Pseudo-classes Used to define special states for elements. button:hover { background-color: lightblue; }
