Advanced CSS Topics - Part 3

1. Float

The float property allows elements to float to the left or right within their container. It's commonly used for page layouts and to wrap text around images.

Key properties related to float include:

  • clear: Prevents elements from wrapping around floated elements. Values include left, right, or both.

Example of a floated element:

float: left;

2. Flex Box

Flexbox is a modern layout system in CSS that helps align and distribute space efficiently within a container, even when the size of its items is dynamic or unknown.

Key Flexbox properties:

  • justify-content: Controls the horizontal alignment of items within the flex container. Options include flex-start, center, and space-between.
  • align-items: Controls the vertical alignment of items. Options include flex-start, center, and stretch.
  • flex-direction: Defines the direction of the main axis (horizontal or vertical). Options include row, row-reverse, column, and column-reverse.
  • flex-wrap: Allows items to wrap if there isn't enough space. Options are nowrap, wrap, and wrap-reverse.

Example of a flex container:

display: flex;

3. Grid Box

Grid Layout is a two-dimensional layout system in CSS. It allows the creation of more complex layouts using rows and columns.

Key Grid properties:

  • grid-template-columns: Defines the number and width of columns. Example: grid-template-columns: 1fr 2fr;.
  • grid-template-rows: Defines the number and height of rows. Example: grid-template-rows: 100px auto;.
  • grid-gap: Adds space between rows and columns. Example: grid-gap: 20px;.

Example of a grid container:

display: grid;

4. Hiding Elements

There are multiple ways to hide elements in CSS depending on whether you want the element to occupy space or not.

Common properties for hiding elements:

  • display: none; Completely removes the element from the layout.
  • visibility: hidden; Hides the element but keeps its space in the layout.
  • opacity: 0; Makes the element transparent, but it remains interactive and part of the layout.

Example of hiding an element:

display: none;

5. Media Query

Media queries allow you to apply CSS rules based on the device's characteristics, such as screen width. This is essential for responsive design.

Key instructions:

  • max-width: Apply styles when the screen width is less than or equal to the defined width.
  • min-width: Apply styles when the screen width is greater than or equal to the defined width.

Example of a media query:

@media (max-width: 600px) { /* styles */ }

6. Styling Font

Font styling includes controlling the appearance of text on the webpage. Common font properties include:

  • font-family: Specifies the font type.
  • font-size: Controls the size of the text.
  • font-weight: Sets the thickness of the font (e.g., bold, normal).
  • color: Defines the text color.

Example of styled text:

font-family: Arial; font-size: 18px;

7. Embedding Web Font

Embedding web fonts allows you to use custom fonts that are not available by default on users' systems.

Example of embedding a web font:

@font-face { font-family: 'MyFont'; src: url('myfont.woff2'); }

8. Flash of Unstyled Text (FOUT)

Flash of Unstyled Text (FOUT) occurs when a web font takes time to load, and the browser displays the fallback font momentarily. You can avoid this using font-display.

Addressing FOUT:

font-display: swap;

9. Font Services

Font services such as Google Fonts provide a wide range of free, ready-to-use fonts that can be embedded into websites.

Using Google Fonts:

@import url('https://fonts.googleapis.com/css2?family=Roboto&display=swap');

10. System Font Stack

A system font stack uses fonts that are already available on users' devices, which improves loading speed and ensures consistency.

Example of using system fonts:

font-family: -apple-system, BlinkMacSystemFont, "Segoe UI", Roboto, "Helvetica Neue", Arial, sans-serif;
